# 일자리 알림핀 요금 정책 (2026 v3 — 추가구매 우선)

> **핵심 방향**: 공고 등록은 **완전 무료**. 적극 노출·PUSH는 **알림핀 크레딧**으로 소진.  
> **수익 모델 (메인 앱)**: 월정액·티어 없음 → **일자리 알림핀 패키지 추가구매**만.  
> **채용 성공 수수료**: 메인 앱 **기본 없음** (`ENABLE_HIRING_COMMISSION=false`). 제휴 채널 빌드에서만 선택 활성화.

## 5. 채용 성공 수수료 (제휴 채널 전용 · 메인 앱 비활성)

| 구분 | 메인 앱 (기본) | 제휴 채널 (`ENABLE_HIRING_COMMISSION=true`) |
|------|----------------|-------------------------------------------|
| 일용직 출근 확인 | **0원** | 코드·결제 화면에 정의된 금액 (레거시: 15,000원/건 참고) |
| 상시직 재직 확인 | 기능 비활성 (`ENABLE_PERMANENT_HIRE=false`) | 월급 5.5% / 30일 (상시직 활성 시) |

- 플래그: `lib/core/config/product_feature_flags.dart` → `enableHiringCommission` (기본 `false`)
- UI·결제·에스컬레이션: `ProductFeatureFlags.isHiringCommissionEnabled`로 게이트
- **문서·FAQ·약관은 메인 앱(수수료 없음) 기준**으로 작성. 제휴 빌드만 별도 안내.
