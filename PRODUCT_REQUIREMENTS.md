# Product Requirements Document (PRD)



## 1. 서비스 개요

**서비스명**: 일자리  

**한 줄 요약**: 일용직 + 상시직을 아우르는 즉시 출근 매칭 플랫폼 (아웃소싱사 대체 게임체인저)



**목표**:

- 기존 모든 기업들의 아웃소싱사 이용을 통한 채용, 비용, 관리 부담을 크게 줄임

- 푸시 강화 + 성과 기반 수수료 하이브리드 모델

- 소규모 자영업자부터 대형 물류·의류·제조 기업까지 모두 사용 가능



## 2. 타겟 사용자

- **구인자 (고용주)**: 소규모 매장 ~ 대형 기업 (다이소, 쿠팡, CJ푸드빌 등)

- **구직자**: 즉시 출근을 원하는 일용직 + 안정적인 상시직 희망자



## 3. 핵심 가치 제안

- **고용주**: 기존 아웃소싱 대비 30~50% 비용 절감 + 노쇼 대폭 감소 + 관리 편의성, 현존하는 구인 서비스들의 게시판형, 점프업, 지원자가 없어도 공고게시만으로도 큰 돈을 선지불해야하는 악질 시스템 대체 

- **구직자**: 빠른 매칭 + 신뢰할 수 있는 고용주 매칭



## 4. 요금제 (2026 — 추가구매 우선)

> **구 모델 폐기**: BASIC / Starter / Growth / Enterprise / 월정액 프리미엄 전부 중단.  
> 상세: [`PUSH_PACKAGE_PRICING.md`](PUSH_PACKAGE_PRICING.md)

### 4.1 기본 플랜 (유일한 무료)

| 항목 | 값 |
|------|-----|
| 공고 등록 | 무료 |
| 푸시 반경 | **1km** |
| 일일 무료 푸시 | **1회** |
| 사업자 보너스 | **5회** (BRN당 1회 지급, 90일) |
| 거점 | **1곳** |
| 월정액 | 0원 |

### 4.2 핵심 상품 — 푸시·거점 패키지

| 항목 | 값 |
|------|-----|
| 정가 | **5,000원** / 패키지 |
| 포함 | 거점 1곳 + 푸시 1회 (**1km**) |

### 4.3 번들

| 상품 | 패키지 수 | 결제액 | 할인 |
|------|-----------|--------|------|
| 10회 팩 | 10 | 45,000원 | 10% |
| 30회 팩 | 30 | 120,000원 | 20% |
| 100회 팩 | 100 | 350,000원 | 30% |

**푸시 전략**: 무료는 1km·1회/일 → 추가 거점·반복 푸시는 **패키지 구매**가 주 경로.

**상시직 수수료**: 재직 확인 기준 30일마다 월 급여 5.5% (기존 유지).

**일용직**: 출근 확인 시 **10,000원** (기존 유지).



## 5. 핵심 기능



### 5.1 구인자 (고용주) 측

- 공고 등록 (일용직 / 상시직 구분)

- 푸시 강화 (거리, 지정 포인트, 추가 푸시)

- Multi-지점 관리 (본사-지역-매장 계층)

- 강력한 ROI 대시보드 (기존 10,000원 대비 절감액 시각화, PDF 리포트)

- 출근 확인 + GPS + ARS 노쇼 방지

- 상시직 재직자 관리 + 건강보험 인증 현황



### 5.2 구직자 측

- 위치 기반 공고 추천

- 지원 → 채팅 → 입사 예정 등록

- 건강보험 자격득실 확인서 간편인증 (상시직)

- 고용주 신뢰 배지 확인

- 출근 체크 (GPS)



### 5.3 공통 / 핵심 정책

- **추가구매 우선**: 기본 1km·1회/일 → 패키지(추가 거점·푸시) 구매 유도

- 사업자등록증 OCR + 업종 검증 (인력공급업 자동 플래그)

- 상시직: 건강보험 인증 → 30일 주기 5.5% 자동 청구



## 6. 기술적 고려사항

- 위치 기반 푸시 (거리 반경)

- 자동 결제 연동

- 건강보험 API (CODEF / Hyphen 등)

- Multi-tenant 구조 (Multi-지점)

- ROI 계산 엔진



## 7. MVP 이후 우선순위 기능

1. ROI 대시보드 고도화

2. Multi-지점 완전 지원

3. 상시직 건강보험 인증 & 자동 청구

4. 구직자 신뢰도 / 고용주 평점 시스템

5. 푸시 최적화 AI
