# 스토어 스크린샷

Play Console / App Store Connect 업로드용 캡처를 이 폴더에 둡니다.

## 권장 캡처 (QC 빌드)

1. 구직자 — 지도 + 공고 마커
2. 구직자 — 공고 상세 · 지원
3. 기업 — 공고 등록 / 알림핀
4. 기업 — 지원자 채팅
5. 더보기 — 사업자 정보 푸터

## 생성 방법

```bash
./run_qc.sh
# Chrome DevTools → 디바이스 툴바 (iPhone 14 Pro 등) → 캡처
```

파일명 예: `01_seeker_map.png`, `02_seeker_apply.png`, …
