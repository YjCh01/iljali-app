## 1. 개인정보의 처리 목적

회사는 다음 목적을 위해 개인정보를 처리합니다.

| 목적 | 상세 |
|------|------|
| 회원 가입·관리 | 본인 확인, 회원 식별, 부정이용 방지 |
| 채용 중개 | 공고 등록·탐색·지원·매칭·채팅 |
| 근태·셔틀 | 출근 확인, 셔틀 예약·알림 |
| 결제·정산 | 알림핀·보유금·유료 서비스 결제 |
| 고객지원 | 문의 응대, 공지 전달 |
| 법적 의무 | 사업자 확인, 분쟁 대응, 관계 법령 준수 |

## 2. 처리하는 개인정보 항목

### 2.1 회원가입·필수

| 구분 | 항목 |
|------|------|
| 공통 | 이름, 이메일, 휴대전화번호, 비밀번호(암호화 저장), 회원유형 |
| 기업회원 | 사업자등록번호, 회사명, 담당자명, 부서, 사업장 주소 |
| 구직자 | 생년월일, 성별, 국적(내·외국인) |

### 2.2 선택

- 희망 근무 지역·직종·가능 일정, 프로필 사진, 경력 요약
- 마케팅 수신 동의 시: 광고성 알림 수신 여부

### 2.3 서비스 이용 과정에서 자동·추가 수집

- 지원·채팅·근태 기록, 알림핀·보유금·결제 내역
- 기기정보(OS, 앱 버전, 기기 식별자), IP, 접속 로그
- 위치정보(GPS·근무지 좌표·지도 이용 시) — 위치기반서비스 이용약관 참조
- 사업자등록증 이미지(OCR·관리자 검토)

### 2.4 민감정보

회사는 원칙적으로 건강·노동조합 등 민감정보를 수집하지 않습니다. 건강보험 자격 확인 등 법령·별도 동의가 필요한 경우에만 최소한으로 처리합니다.
