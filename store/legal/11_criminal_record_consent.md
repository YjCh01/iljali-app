# 범죄경력조회 동의서

**시행일:** 2026년 9월 1일

---

## 1. 수집·이용 목적

채용 절차에서 기업회원(또는 채용 담당자)이 요청하는 경우, 구직자의 범죄경력 확인(법령이 허용하는 범위)

## 2. 수집 항목

| 항목 | 내용 |
|------|------|
| 성명, 생년월일 | 본인 확인 |
| 조회 결과 | 범죄경력 유무(법령 허용 범위) |
| 조회 경로 | 경찰청 범죄경력회보서 발급 또는 법령이 정한 전자 조회 |

## 3. 보유·파기

채용 절차 종료(합격·불합격·지원 취소) 후 **30일** 이내 파기합니다. 채용이 확정된 경우 관련 법령이 정한 기간 동안 보관할 수 있습니다.

## 4. 동의 거부

동의를 거부할 수 있으나, 거부 시 해당 채용 절차 진행이 제한될 수 있습니다.
