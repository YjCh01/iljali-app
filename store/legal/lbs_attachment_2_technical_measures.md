## 2. 암호화 및 방화벽

| 항목 | 내용 |
|------|------|
| 전송(구간) 암호화 | **HTTPS / TLS 1.2 이상**, 인증서 SHA-256 |
| 적용 도메인 | `https://iljari.app`, `https://api.iljari.app` |
| 저장 | DB 접근통제·최소권한, 민감정보 보호 (컬럼 암호화는 단계적 강화) |
| 방화벽 | 네이버클라우드(NCP) **ACG** — 필요 포트(80/443 등)만 개방 |
| 서버 접근 | SSH 키 기반, 무단 접근 차단 |
| 증빙 | SSL 인증서 상세(유효기간 1개월 이상), NCP ACG 인바운드 화면 |

## 6. 이용·제공사실 자동 기록

위치정보 수신·이용 시 대상·취득경로·제공서비스·제공받는 자(해당 시)·이용일시(초)를  
위치정보시스템이 **전자적으로 자동 기록·보존**한다.
