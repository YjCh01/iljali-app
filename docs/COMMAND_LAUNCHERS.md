# iljari.app 런처

Finder 정렬 순서: **실서비스/개발 → 앱/웹 → 회원분류 → (플랫폼)**

## 주소

| | URL |
|--|-----|
| 통합 웹 (맵 먼저) | https://iljari.app/ |
| 어드민 웹 | https://iljari.app/admin/ |
| API | https://api.iljari.app |

> `/corporate/` 는 `/` 로 리다이렉트. 기업/개인 선택은 맵에서 로그인·가입 시 게이트웨이.

## 웹 (6)

| 파일 | 동작 |
|------|------|
| `실서비스_웹_개인회원` | 배포 + https://iljari.app/ (통합 site) |
| `개발_웹_개인회원` | 맥 Chrome hot reload |
| `실서비스_웹_기업회원` | 배포 + site (기업 빌드 플래그) |
| `개발_웹_기업회원` | 맥 Chrome |
| `실서비스_웹_어드민` | 배포 + /admin/ |
| `개발_웹_어드민` | 맥 Chrome |

## 앱 — Android / iOS (12)

| | 안드로이드 실서비스 | 안드로이드 개발 | 아이폰 실서비스 | 아이폰 개발 |
|--|--|--|--|--|
| 개인회원 | `실서비스_앱_개인회원_안드로이드` | `개발_앱_개인회원_안드로이드` | `실서비스_앱_개인회원_아이폰` | `개발_앱_개인회원_아이폰` |
| 기업회원 | `실서비스_앱_기업회원_안드로이드` | `개발_앱_기업회원_안드로이드` | `실서비스_앱_기업회원_아이폰` | `개발_앱_기업회원_아이폰` |
| 어드민 | `실서비스_앱_어드민_안드로이드` | `개발_앱_어드민_안드로이드` | `실서비스_앱_어드민_아이폰` | `개발_앱_어드민_아이폰` |

- **실서비스** = API `https://api.iljari.app` · 네이티브 `flutter run`
- **개발** = 맥 로컬 API `127.0.0.1:8000`

## 도구

| 파일 | 용도 |
|------|------|
| `도구_실서비스한방배포` | **API + 웹(site·admin) 한 번에** — SSH 1회 |
| `도구_웹전체배포` | site + admin (웹만) |
| `도구_API배포` | FastAPI → api.iljari.app (가입·어드민·OCR/국세청) |
| `도구_사이트완료` | SSL + 개인 루트 배포 |
| `도구_브라우저열기` | https://iljari.app |
| `도구_전체점검` | DNS·HTTPS 점검 |
